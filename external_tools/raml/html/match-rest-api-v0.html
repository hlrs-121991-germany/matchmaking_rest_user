/questions/{id} get put delete

get

Retrive specific Question. ID is the URI parameter in string, and the value is an unique integer (Question Identity).

put

Update the Question details partially or completely. ans-add, ans-remove is the string value to list answer id. Eg. "1,3,4,5,7,8".

delete

Delete the Question.
